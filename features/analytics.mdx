### Predictive Metrics

Real-time metrics enhanced with AI predictions:

<Tabs>
  <Tab title="Demand Forecasting">
    **7-30 Day Demand Predictions**
    
    AI analyzes historical sales, seasonal patterns, weather, local events, and social media trends to forecast demand with 98% accuracy.
    
    ```javascript
    // Example: Get demand forecast
    const forecast = await salsa.ai.analytics.getDemandForecast({
      location: "kitchen_1",
      horizon_days: 14,
      recipes: ["house_salsa", "spicy_chipotle"],
      confidence_level: 0.95
    });
    
    forecast.predictions.forEach(day => {
      console.log(`${day.date}: ${day.predicted_demand} gallons`);
      console.log(`  Confidence: ${day.confidence_interval[0]} - ${day.confidence_interval[1]}`);
      console.log(`  Key factors: ${day.primary_factors.join(", ")}`);
    });
    ```
    
    **What influences demand:**
    - Historical sales patterns (40%)
    - Day of week and seasonality (25%)
    - Weather conditions (15%)
    - Local events and holidays (10%)
    - Social media and reviews (10%)
    
    <Info>
      Demand forecasts update daily at 3 AM and are available in the dashboard by 6 AM.
    </Info>
  </Tab>
  
  <Tab title="Quality Predictions">

### Smart Alerts

AI-powered intelligent alerting system that learns what's important to you.

```javascript
// Example: Configure AI-powered alerts
const alert = await salsa.ai.analytics.createSmartAlert({
  type: "anomaly_detection",
  metric: "production_efficiency",
  sensitivity: "medium",  // low, medium, high
  actions: ["notify_manager", "create_ticket"],
  recipients: ["manager@restaurant.com"],
  schedule: {
    active_hours: "6am-11pm",
    days: ["monday", "tuesday", "wednesday", "thursday", "friday"]
  }
});
```

**Alert Types:**
- **Anomaly Detection**: Automatically detects unusual patterns
- **Threshold Alerts**: Trigger when metrics cross defined thresholds
- **Predictive Alerts**: Warn before issues occur
- **Trend Alerts**: Notify about significant trend changes

<Note>
  **Smart Alert Learning**: The system learns which alerts you act on and automatically adjusts sensitivity to reduce false positives.
</Note>

## Predictive Analytics

### Demand Forecasting

Our AI-powered demand forecasting uses multiple machine learning models for industry-leading accuracy.

**Forecasting Models:**
- **ARIMA**: Time series analysis for baseline predictions
- **Prophet**: Seasonal decomposition and trend analysis
- **Neural Networks**: Deep learning for complex patterns
- **Ensemble**: Combined predictions for optimal accuracy

```graphql
# Example: GraphQL query for AI forecast
query GetAIForecast {
  forecast(location: "loc_123", days: 14) {
    predictions {
      date
      recipes {
        name
        predictedDemand
        confidenceInterval {
          lower
          upper
        }
        confidence
      }
      factors {
        weather {
          condition
          temperature
          impact_percentage
        }
        events {
          name
          distance_miles
          expected_attendance
          impact_percentage
        }
        seasonality {
          day_of_week_factor
          seasonal_factor
          holiday_factor
        }
      }
    }
    accuracy_metrics {
      mape
      rmse
      bias
    }
  }
}
```

**Forecast Accuracy:**
- **7-day forecast**: 98% accuracy (MAPE < 2%)
- **14-day forecast**: 95% accuracy (MAPE < 5%)
- **30-day forecast**: 90% accuracy (MAPE < 10%)

<Tip>
  Forecast accuracy improves over time as AI learns your specific patterns. Most customers see 5-10% accuracy improvement within 60 days.
</Tip>
