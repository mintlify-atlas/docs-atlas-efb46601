## Getting Started

<Steps>
  <Step title="Connect IoT Testing Equipment">
    Install pH meters, viscosity sensors, color sensors, and temperature probes in your production areas.
    
    Our hardware partners offer complete testing kits designed specifically for salsa production. Each sensor connects wirelessly and automatically syncs data to the cloud.
    
    <Info>
      **Recommended Setup**: Install sensors at 3 key checkpoints - ingredient prep, mixing, and final packaging.
    </Info>
  </Step>
  <Step title="Enable AI Quality Prediction">
    Activate AI predictions in **Quality Control > AI Features** and import at least 90 days of historical quality data.
    
    ```javascript
    // Example: Enable AI quality prediction
    const aiQuality = await salsa.ai.quality.enable({
      location: "kitchen_1",
      historical_batches: 200,
      prediction_model: "advanced"
    });
    ```
    
    The AI requires historical data to learn what makes a high-quality batch in your operation.
  </Step>
  <Step title="Configure Quality Thresholds">
    Set your minimum quality scores and alert thresholds in **Quality Control > Standards**. AI will auto-adjust these over time based on your actual production.
    
    <Tip>
      Start with industry standard thresholds (7.0/10 minimum). The system will recommend adjustments after analyzing your first 50 batches.
    </Tip>
  </Step>
  <Step title="Set Up Computer Vision">
    Position cameras at key production checkpoints and run the calibration routine via the mobile app.
    
    The system will guide you through optimal camera placement for maximum coverage.
  </Step>
  <Step title="Monitor & Improve">
    Review AI quality insights daily and let the system learn your production patterns to improve predictions over time.
    
    Most customers see prediction accuracy improve from 85% to 95%+ within 60 days.
  </Step>
</Steps>

## AI Quality Prediction

### Predictive Quality Scoring

Our AI analyzes ingredient quality data and production conditions in real-time to predict final quality scores before batch completion.

<CardGroup cols={2}>
  <Card title="Ingredient Analysis" icon="flask">
    Analyzes ingredient quality data and production conditions in real-time to
    predict final quality scores with 95% accuracy.
    
    Factors analyzed:
    - Ingredient freshness scores
    - Supplier quality history
    - Storage temperature compliance
    - Days since delivery
  </Card>
  <Card title="Early Warning System" icon="bell">
    Provides early warnings for potential issues before they impact your batch,
    giving you time to intervene.
    
    Average warning time: 15-30 minutes before completion.
  </Card>
</CardGroup>

```javascript
// Example: Get AI quality prediction
const prediction = await salsa.ai.quality.predict({
  batch_id: "BATCH-2024-001",
  current_stage: "mixing",
  include_recommendations: true
});

console.log(`Predicted quality: ${prediction.score}/10`);
console.log(`Confidence: ${prediction.confidence}%`);

if (prediction.score < 8.0) {
  console.log(`⚠️  Warning: ${prediction.warning_message}`);
  console.log(`Recommendation: ${prediction.recommended_action}`);
}

// Example response
{
  "batch_id": "BATCH-2024-001",
  "predicted_score": 7.2,
  "confidence": 92,
  "current_stage": "mixing",
  "completion_percentage": 65,
  "warning_message": "Predicted score below target (8.0)",
  "recommended_action": "Increase mixing time by 2 minutes to improve consistency",
  "factors": {
    "ingredient_quality": 8.5,
    "mixing_temperature": 6.8,  // Below optimal
    "mixing_time": 7.0,          // Below optimal
    "ph_level": 8.2
  },
  "time_to_intervention": "18 minutes"
}
```

<Warning>
  **Act Quickly**: When AI predicts quality issues, you typically have 15-30 minutes to make adjustments before the batch is complete.
</Warning>

## Quality Scoring System

### AI-Enhanced Scoring

Our quality scoring system combines automated sensor data with manual taste testing for comprehensive quality assessment.

```javascript
// Example: Record quality with AI assistance
const qualityCheck = await salsa.ai.quality.recordScore({
  batch_id: "BATCH-2024-001",
  automated_scores: {
    visual_ai: 9.2,        // Computer vision score
    ph_sensor: 8.8,        // pH within optimal range
    viscosity: 9.0,        // Consistency score
    temperature: 10.0,     // Perfect temp compliance
    color: 8.5             // Color consistency score
  },
  manual_scores: {
    taste: 9.5,            // Manual taste test
    aroma: 9.0,            // Smell test
    appearance: 9.2        // Visual inspection
  },
  ai_analysis: true,
  taster_id: "user_123"
});

console.log(`Overall Quality Score: ${qualityCheck.overall_score}/10`);
console.log(`Automated: ${qualityCheck.automated_average}/10`);
console.log(`Manual: ${qualityCheck.manual_average}/10`);
```

**Score Breakdown:**
- **Automated Scores (60% weight)**: Objective measurements from sensors
- **Manual Scores (40% weight)**: Human sensory evaluation
- **Overall Score**: Weighted average of all factors

<Tip>
  Maintain at least 3 trained taste testers to ensure consistency in manual scoring. AI tracks tester reliability and can flag outlier scores.
</Tip>

## Critical Quality Alerts

<Warning>
  **Critical Quality Alerts**: AI-detected safety issues trigger immediate
  production halts and manager notifications.
</Warning>

When the system detects a critical quality or safety issue:

1. **Immediate Production Halt**: Affected equipment stops automatically
2. **Manager Notification**: SMS and email alerts sent to all managers
3. **Mobile App Alert**: Push notifications to all relevant staff
4. **Incident Documentation**: Automatic logging of all details
5. **Corrective Action Workflow**: Guided resolution process

**Critical Issues Include:**
- Temperature excursions beyond safe ranges
- Contamination detected by vision systems
- pH levels indicating spoilage
- Equipment malfunction affecting food safety
- Allergen cross-contamination risks
